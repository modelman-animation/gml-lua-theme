# GameMaker Lua Theme

GameMaker-inspired dark theme for Lua and Moho scripting

---

## Features

- 🎮 **GameMaker-inspired colors** – Syntax highlighting that feels like GML
- 🎨 **Dark theme optimized** – Easy on the eyes for long coding sessions
- 🔵 **Moho Lua support** – Perfect for Moho scripting
- 🚀 **Familiar experience** – Reduce the learning curve when transitioning to Moho

## Why This Theme?

If you're a GameMaker developer starting with Moho or Lua, this theme makes the transition smoother by using colors you already know and love from the GML editor.

## Installation

1. Open VS Code
2. Go to Extensions (`Ctrl+Shift+X`)
3. Search for **"GameMaker Lua Theme"**
4. Click **Install**
5. Go to **File → Preferences → Color Theme**
6. Select **"GameMaker Lua Theme"**

## Recommended Extensions

For the best experience, install:

- [Moho Script Helper](https://marketplace.visualstudio.com/items?itemName=Modelman.moho-script-helper) – Lua autocomplete and stubs for Moho 14.3+ scripting
